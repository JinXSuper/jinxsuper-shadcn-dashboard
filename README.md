## Features

| | Feature | Description |
|---|---|---|
| ⚡ | **Next.js 16 + React 19** | Built on the latest App Router with full React 19 support |
| ✦ | **Glassmorphism UI** | Custom `GlassSurface` components with Three.js light pillar effects |
| ♟ | **shadcn/ui + Radix** | Accessible, production-ready components pre-configured |
| 📊 | **Interactive Charts** | Real-time Recharts dashboards with live metric cards |
| 🗂 | **Responsive Navigation** | Smooth sidebar collapse/expand with mobile-friendly transitions |
| 🎞 | **Framer Motion** | Buttery animations and micro-interactions |

## Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/JinXSuper/jinxsuper-shadcn-dashboard.git
cd jinxsuper-shadcn-dashboard
```

**2. Install dependencies**
```bash
npm install
```

**3. Run the development server**
```bash
npm run dev
```

**4. Build for production**
```bash
npm run build
```
