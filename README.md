# JinXSuper Dashboard Template (Vercel-like UI)

A stunning, professional-grade admin dashboard template built with the latest 2025 tech stack. Inspired by Vercel's clean aesthetic, this template features glassmorphism, smooth animations, and a focus on visual excellence.

## 🚀 Features

- **Built with Next.js 16.1 & React 19.2**
- **Modern Styling**: Tailwind CSS v4 for lightning-fast, modern UI development.
- **Glassmorphism & Light Pillar Effects**: Premium visual design with custom `GlassSurface` components and Three.js light effects.
- **Radix UI & shadcn/ui**: Accessible, high-quality components pre-configured.
- **Responsive Navigation**: Smooth sidebar transitions (collapse/expand) and mobile-friendly layouts.
- **Interactive Dashboards**: Real-time charts (Recharts), data tables, and interactive metric cards.
- **Open Source**: Built by **JinXSuper Developer** for the community.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4
- **Components**: shadcn/ui & Radix UI
- **Icons**: Lucide React
- **Animations**: Framer Motion / CSS Transitions
- **Charts**: Recharts
- **Background Effects**: Three.js

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jinxsuper/dashboard-template.git
   cd dashboard-template
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
